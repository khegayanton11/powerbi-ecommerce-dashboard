# Power BI E-commerce Analytics Dashboard

## Описание проекта

Компания столкнулась со снижением продаж и поставила задачу выявить причины падения выручки, определить проблемные зоны в логистике, работе складов, клиентской базе и эффективности персонала.

Для решения задачи был разработан интерактивный аналитический отчет в Power BI на основе данных интернет-магазина за период более одного года.

## Используемые инструменты

* Power BI
* DAX
* Power Query
* Data Modeling

## Источники данных

В проекте использовались данные:

* Клиенты
* Заказы
* Продажи
* Доходы
* Склады
* Остатки товаров
* Сотрудники
* Логистика и доставка
* Справочники товаров и статусов заказов

## Построение модели данных

* Выполнен анализ структуры данных.
* Создана модель данных со связями между таблицами.
* Настроены отношения между фактами и справочниками.
* Реализована схема для последующего анализа показателей бизнеса.

## Реализованные DAX-меры

* Общая выручка
* Прибыль
* Маржинальность
* Средний чек
* Доля возвратов
* Выручка по складам
* Выручка по регионам

## Структура отчета

### 1. Продажи и прибыль

* Динамика продаж
* Анализ прибыли
* Структура выручки по категориям товаров
* Ключевые финансовые показатели

### 2. Логистика и склады


* Остатки товаров
* Выручка по складам
* Анализ логистических затрат
* Эффективность складов

### 3. Клиенты и регионы

* География клиентов
* Активность по регионам
* Динамика новых клиентов
* Анализ клиентской базы

### 4. Сотрудники и эффективность

* Фонд оплаты труда
* Количество сотрудников
* Производительность складов
* Анализ эффективности персонала

### 5. Аналитические выводы

В отчете выполнен анализ:

* причин снижения продаж;
* логистических затрат по регионам;
* доли возвратов;
* сезонности продаж;
* эффективности работы персонала.

## Основные результаты

* Выявлены ключевые факторы снижения продаж.
* Определены регионы с повышенными логистическими затратами.
* Выполнен анализ возвратов.
* Выявлена сезонность спроса.
* Найдены потенциальные точки роста бизнеса.
